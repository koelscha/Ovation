RASAtest
==================
##Files
config_spacy.json:  configuation of RASA training
RASAtest.py:  RASA training and execution & entity extraction by spacy
demo-rasa.json: Training data for RASA

##Description
Calling RASA & spaCy with Python script
	- intent analysis is done by RASA
	- entity extraction is done by spacy

##Requirement
	- rasa_nlu (bleeding edge version)
	- python 2.7
	- spacy

For spaCy
